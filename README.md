# Law Firm Website Design

React + Vite site exported from [Figma Make](https://www.figma.com/design/yc1l3TnN6eWXEif9u3Qxwj/Law-Firm-Website-Design).

## Prerequisites

- Node.js 18+ (20+ recommended)
- npm (or pnpm)

## Local development

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).

## Production build

```bash
npm run build
npm run preview
```

The static site is output to `dist/`. Serve that folder with any static host (Netlify, Vercel, GitHub Pages, etc.).

## Routes

| Path | Page |
|------|------|
| `/` | Home |
| `/login` | Login |
| `/signup` | Sign up |
| `/client` | Client landing |
| `/lawyer` | Lawyer dashboard |
| `/admin` | Admin dashboard |
| `/conversations` | Conversations |
